SpaceX GraphQL Viewer

This is a personal project I created to help me learn GraphQL/Apollo and Firebase hosting.

By clicking on the three buttons on the navbar, you can switch between viewing this page, past SpaceX launches, and any future launches. Clicking on an entry in the past launches tab will take you to a details page, but this is not implemented for future launches since the details returned by the API is spotty sometimes.

I chose this API since it was clean and easy to use, plus I recently started following the innovations and technological at SpaceX. The data is publicly available via this API, but please not I am not affiated with SpaceX in any capacity. This was just for fun and educational purposes only.
